# OFFICE - FollowUp Prompt Pack

Use these flows with your Notion Office system:

- `A -> D`: Task progress/problem updates in `Work Tasks`
- `B -> E`: End-of-day execution summary in `Daily Work Log`
- `C -> F`: Reusable lesson in `Task Knowledge`

## Quick chooser
- Working on an active task now: `A -> D`
- Recording what happened today: `B -> E`
- Capturing reusable validated lesson: `C -> F`

## Notion field alignment (validated)
- Work Tasks:
  - `Task name`
  - `Status-Development`
  - `Blocked`
  - `Note`
  - `Knowledge Page`
- Daily Work Log:
  - `Name`
  - `Date`
  - `Summary`
  - `Tasks Worked`
  - `Blockers`
  - `Next Actions`
- Task Knowledge:
  - `Title`
  - `Task`
  - `Knowledge Type`
  - `Merge Status`

All write prompts assume:
- Search before write
- Prefer Update over Create
- Stop and ask if multiple matches
- No schema/view changes
- No unrelated edits
